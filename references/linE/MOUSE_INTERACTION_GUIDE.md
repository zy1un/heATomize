# 鼠标交互功能使用指南

## ✅ 已实现的功能

### 1. **鼠标悬停高亮** 🖱️
- **功能描述：** 当鼠标移动到格子上时，格子会高亮显示。
- **视觉效果：** 格子边框加粗。
- **操作方法：** 只需移动鼠标到任意格子上方。

### 2. **点击选择球** 🎯
- **功能描述：** 点击有球的格子可以选中该球。
- **视觉效果：** 选中的球会开始**呼吸缩放动画**。
- **操作方法：** 用鼠标左键点击。

### 3. **取消与切换选择** 🔄
- **操作方法：** 
  - 点击已选中的球 -> 取消选择。
  - 点击另一个有球的格子 -> 切换选中目标。

### 4. **球体移动 (已实现)** 🚀
- **功能描述：** 选中球后，点击空格子，系统将自动寻找最短路径并演示移动过程。
- **动画表现：** 
  - **路径高亮**：球在移动过程中，走过的路径格子会持续高亮显示。
  - **逐格动画**：球会沿路径平滑滚动。
  - **愤怒跳动**：如果目标位置不可达，球会原地剧烈跳动（跺脚）提示寻路失败。

### 5. **商店交互** 🛒
- **打开方式**：点击侧边栏 `SHOP` 按钮。
- **购买流程**：在商店面板中点击商品，分数足够则扣款成功。
- **特种投放**：购买彩虹球后，光标会自动携带彩虹球，点击棋盘空格子即可投放。

### 6. **存读档面板** 💾
- **打开方式**：点击 `LOAD` 或 `SAVE`。
- **操作**：在弹出的浮层中选择 1-3 号槽位，带缩略信息的交互。

## 🎮 操作小技巧
- **快捷取消**：在彩虹球投放模式下，点击棋盘外的区域可取消购买并自动退款。
- **平滑体验**：消除动画进行中时，系统会自动屏蔽点击干扰，确保逻辑严谨。

## 🔧 技术实现方案
- **寻路算法**：BFS (Breadth First Search) 实现于 `MapPathfinding.cpp`。
- **坐标转换**：利用 `GridSystem` 标准化像素与网格的映射。
- **事件过滤**：主循环 `Game::run` 优先处理 UI 层（商店、存档面板）的点击，防止事件穿透到棋盘。
