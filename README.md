# heATomzie

Godot prototype for a heat-driven cluster elimination game.

## Current Scope

The project is in early setup. The first playable target is:

- 9x9 board
- movable heat balls
- turn-based spawning
- heat update
- cluster elimination
- aftershock chain reactions

## Project Layout

- `scenes/` Godot scenes
- `scripts/` gameplay scripts
- `assets/` game assets
- `data/` tuning data and rule notes
- `ui/` UI scenes and scripts
