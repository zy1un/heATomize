## 设计实验 (Design Experiments)

- [ ] 像素化计分板素材方向：
  - [ ] 优先用 Godot UI、像素字体、简单发光 shader 或程序化小贴图实现。
  - [ ] 暂不依赖完整 AI 生成大图作为 UI 主素材。
  - [ ] 如果需要生成素材，优先生成小尺寸像素风 sprite sheet：数字管、边框角块、电火花、小粒子、过载闪烁帧。
- [ ] 连击加速 (Chain acceleration)：保持刚移动后的初步动画温和易读，然后在级联发生时逐步加快链式反应速度。
  - 基础思路：第一波消除正常速度，第 2 波快约 1.25 倍，第 3 波快约 1.5 倍，极限限制在 2.5 倍以保持可读性。
  - 将加速应用到热量传导动画、阶段暂停、消除收缩和余震传导，作为一个全局节奏乘数。
  - 如果加入了音频和屏幕震动，可搭配更强烈的音调和视觉反馈。
  - 战术模式保持相对稳健；混沌模式可使用稍激进的加速曲线。
- [ ] 经过几次实际游玩测试后，对比验证战术模式和混沌模式的长期体验。
