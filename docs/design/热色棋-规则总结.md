# 热色棋：规则总结

> 版本：v0.4（2026-03-30）

---

## 一、基本设定

| 项目 | 设定 |
|------|------|
| 棋盘 | $9 \times 9$ 正方网格 |
| 邻域 | **4-邻域**（上下左右），全游戏统一 |
| 球的唯一属性 | 温度 $\tau \in \{1,2,3,4,5\}$ |
| 视觉映射 | 1 🔵蓝 → 2 🟢绿 → 3 🟡黄 → 4 🟠橙 → 5 🔴红 |
| 结束条件 | 玩家无法移动任何球（无可达空格路径）时判负 |

---

## 二、每回合流程

```
① 玩家移动一个球（沿空格路径，4-邻域可达）
   ※ 若所有球均无可达空格路径，判负，游戏立即结束
        ↓
② 随机生成 min(3, 空格数) 个球，温度 τ ∈ {1,2,3} 随机，位置随机
        ↓
③ 系统更新：
     Phase 1: 温度变化（所有球同时计算）
     Phase 2: 消除检测
     Phase 3: 余震
     ← 循环 Phase 1~3，直到无新消除
```

---

## 三、温度变化（Phase 1）

**一条规则，三种情况，每球每回合最多变化 ±1：**

| 条件 | 变化 |
|------|------|
| 存在任意邻居比自己**热** | $\tau \mathrel{+}= 1$ |
| 所有邻居都比自己**冷**（或无邻居） | $\tau \mathrel{-}= 1$ |
| 其余情况（与邻居同温，或冷热都有但无更热的） | 不变 |

> 注：所有球**同时**用旧值计算，再统一更新。温度截断在 $[1, 5]$。

**直觉：** 在热源旁边的球会被烤热；孤立或周围全是冷球的热球会慢慢冷却；一群同温的球彼此不影响。

---

## 四、消除检测（Phase 2）

在所有**同温度**的球中，找 4-邻域连通体。

若连通体大小 $|C| \geq \text{threshold}(\tau)$，则整体消除。

### 消除阈值

| $\tau$ | 颜色 | 阈值 | 含义 |
|--------|------|------|------|
| 1 | 🔵 蓝 | 7 | 最稳定 |
| 2 | 🟢 绿 | 6 | 稳定 |
| 3 | 🟡 黄 | 5 | 中等 |
| 4 | 🟠 橙 | 4 | 不稳定 |
| 5 | 🔴 红 | 3 | 极不稳定 |

> 核心逻辑：**越热越不稳定，越少的球凑在一起就会爆。**

---

## 五、余震（Phase 3）

消除后，所有被消除格子的邻居（非空）：

$$\tau_{\text{邻居}} \mathrel{+}= 1 \quad \text{（上限 5）}$$

余震后返回 Phase 1 重新计算温度，再进行消除检测，形成**级联反应**。

---

## 六、核心涌现特性

### 定时炸弹效应

孤立小团体会缓慢升温，最终在高温阶段以更少的数量爆发：

| 集群大小 | 爆发温度 | 感觉 |
|---------|---------|------|
| ≥ 7 | 🔵 蓝（原地就爆） | 即时清除 |
| 5~6 | 🟢🟡 绿/黄 | 短引信 |
| 4 | 🟠 橙 | 中期威胁 |
| 3 | 🔴 红 | 长引信定时炸弹 |
| 2 | 永不自爆 | 未爆弹，等第三个 |

### 热波传播

余震制造热核 → 热传导向外扩散 → 外圈同温球逐渐达到阈值 → 再次消除 → 新余震……形成肉眼可见的同心温度波。

### 路径障碍

移动需要 4-邻域可达路径。棋盘上的球群天然形成障碍，玩家必须绕路或先清除障碍才能移动，隔断与重塑路径本身是博弈内容。

---

## 七、玩家策略空间

| 策略 | 描述 |
|------|------|
| 主动引爆 | 将同温球移到一起触发消除，清出空间 |
| 引线布局 | 安排球的位置引导余震传播方向 |
| 隔热防火 | 移开球切断传导路径，防止不想要的级联 |
| 未爆弹管理 | 保留 2 个红球等待第三个到位，引爆最大级联 |
| 路径博弈 | 利用或制造障碍，限制对手（如为多人版）或管理局面 |

## 八、待确定参数

| 参数 | 当前值 | 备注 |
|------|--------|------|
| 各温度消除阈值 | 7/6/5/4/3 | 需模拟验证 |
| 每回合生成球数 | 3 | 可根据节奏调整 |
| 新球温度范围 | $\tau \in \{1,2,3\}$ 随机 | 不直接生成高温球 |
| 余震强度 | 统一 +1 | 可考虑高温消除余震更强 |

## 九、设计哲学

```
一个状态维度：温度（= 颜色）
一条温度规则：有更热的邻居就升，比所有邻居都热就降，否则不变
一个消除规则：同温连通体够大就爆（越热门槛越低）
一个传播机制：余震给邻居 +1

→ 涌现出：热波扩散、定时炸弹、级联反应、路径博弈
```
